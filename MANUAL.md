## 2. インストール・起動手順

### 2-1. 初回セットアップ

ターミナルを開き、プロジェクトフォルダへ移動して依存パッケージをインストールします。

```bash
# プロジェクトフォルダへ移動
cd /Volumes/CT1000P3SSD/musicplayer

# 依存パッケージのインストール（初回のみ）
npm install
```

### 2-2. アプリの起動

```bash
# 通常起動
npm start

# 開発モード（デベロッパーツール有効）
npm run dev
```

### 2-3. アプリの終了

- ウィンドウ左上の **赤いボタン（●）** をクリック
- または `Command + Q`

## 4. 楽曲の追加方法

### 4-1. ドラッグ＆ドロップ（推奨）

1. Finder でオーディオファイル（またはフォルダ）を選択
2. アプリの **左パネルのドロップゾーン**（「MP3 / M4A ファイルをここにドロップ」と表示されている箇所）へドラッグ
3. 自動的にメタデータ（タイトル・アーティスト・カバーアート等）が読み込まれプレイリストに追加されます

> **💡 ヒント:** フォルダごとドロップすると、フォルダ内のオーディオファイルをすべて一括追加できます。

### 4-2. 対応フォーマット

| 拡張子 | フォーマット |
|--------|------------|
| `.mp3` | MP3 |
| `.m4a` | MPEG-4 Audio |
| `.aac` | AAC |
| `.flac` | FLAC (可逆圧縮) |
| `.wav` | WAV |
| `.ogg` | Ogg Vorbis |

## 5. 再生コントロール

### 5-1. ボタン一覧（中央パネル）

| ボタン | 機能 |
|--------|------|
| ▶ / ❚❚ | 再生 / 一時停止 |
| ⏮ | 前の曲（再生3秒以上経過時は曲の先頭へ） |
| ⏭ | 次の曲 |
| ↺ `10` | 10秒巻き戻し |
| ↻ `10` | 10秒早送り |
| 🔁 | リピート ON/OFF（アクティブ時は白く点灯） |
| 🔀 | シャッフル ON/OFF（アクティブ時は白く点灯） |

### 5-2. シークバー操作

- シークバーをクリックまたはドラッグして再生位置を変更できます
- シークバーの背景部分をクリックしても同様に移動可能

## 6. 音量・速度の調整

### 6-1. 音量

- **起動時:** macOS のシステムボリューム値が自動的に反映されます
- 中央パネル下部の **VOLUMEスライダー** で手動調整可能
- 範囲: 0% ～ 100%

### 6-2. 再生速度

**プリセットボタン:**

| ボタン | 速度 |
|--------|------|
| 0.5x | 0.5倍速（スロー再生） |
| 0.75x | 0.75倍速 |
| 1.0x | 標準速度（デフォルト） |
| 1.25x | 1.25倍速 |
| 1.5x | 1.5倍速 |
| 2.0x | 2倍速 |

**カスタム速度の設定:**

1. 数値入力欄に `0.1` ～ `4.0` の値を入力
2. `SET` ボタンをクリック（または `Enter` キー）

## 7. レベルメーター

### カラー表示

| 色 | レベル範囲 | 意味 |
|----|-----------|------|
| 🔵 青 | 0 〜 70% | 適正レベル |
| 🟡 黄 | 70 〜 90% | やや大きいレベル |

> **⚠️ 注意:** 赤ゾーンが頻繁に点灯する場合は音量を下げてください。

## 9. イコライザー（EQ）

右パネルの **「EQ」タブ** で設定します。

### 9-1. EQ ON/OFF

- 右上のトグルスイッチでイコライザー全体をON/OFFできます
- **起動時はROCKプリセットがON状態で適用されます**

### 9-2. プリセット一覧

| プリセット | 特徴 |
|-----------|------|
| **Flat** | フラット（補正なし） |
| **Rock** | 低域と高域を強調（デフォルト） |
| **Pop** | 中域を強調 |
| **Jazz** | 低域と高域をやや強調 |
| **Classical** | 低域と高域を強調（繊細な表現） |
| **Voice** | 中高域（声）を強調 |
| **Bass↑** | 低域を大幅強調 |
| **Heavy Metal** | 低域・中高域を強烈に強調 |

### 9-3. 5バンドEQスライダー

各スライダーを上下に動かすことで手動調整できます。

| バンド | 周波数 | 対象音域 |
|--------|--------|---------|
| Band 1 | 60 Hz | 超低域（サブベース） |
| Band 2 | 250 Hz | 低域（ベース） |
| Band 3 | 1 kHz | 中域（ボーカル等） |
| Band 4 | 4 kHz | 中高域（存在感） |
| Band 5 | 16 kHz | 高域（空気感） |

- 範囲: **-12dB ～ +12dB**
- スライダーを手動操作するとプリセット選択が解除されます

### 9-4. EQリセット

「リセット」ボタンをクリックするとすべてのバンドが 0dB（Flat）に戻ります。

## 10. Spatial Audio（空間音響）

右パネルの **「SPATIAL」タブ** で設定します。

### 10-1. 空間音響モード

| モード | 説明 |
|--------|------|
| **OFF** | Spatial Audio無効 |
| **Concert** | コンサートホール風の残響 |
| **Theater** | 映画館風の残響 |
| **Room** | 部屋の中にいるような残響 |
| **Studio** | スタジオ録音風（少なめの残響） |
| **Outdoor** | 屋外の開放感のある広い残響 |

### 10-2. パラメーター調整

| パラメーター | 説明 | 範囲 |
|------------|------|------|
| **リバーブ** | 残響の強さ | 0% ～ 100% |
| **ステレオ幅** | 左右の広がり | 0% ～ 200% |
| **コンプレッサー** | 音量の均一化 | 0% ～ 100% |

## 11. プレイリスト操作

### 11-1. 楽曲の選択・再生

- プレイリスト内の楽曲をクリックすると即座に再生開始
- 再生中の楽曲はアニメーションインジケーターで表示されます

### 11-2. シャッフル

左パネル下部の **「🔀 シャッフル」** ボタンでプレイリストの順序をランダムに並び替えます。

### 11-3. プレイリストのクリア

左パネル下部の **「🗑 クリア」** ボタンでプレイリストをすべて削除します。

> **⚠️ 注意:** クリアすると再生も停止されます。元に戻すことはできません。

## 14. トラブルシューティング

### 音が出ない

1. macOS のシステム音量が0になっていないか確認
2. アプリ内のVOLUMEスライダーが0でないか確認
3. 一度停止してから再度再生ボタンをクリック
4. アプリを再起動（`Command + Q` → `npm start`）

### ファイルが追加されない

- 対応フォーマット（MP3/M4A/AAC/FLAC/WAV/OGG）であるか確認
- ファイルパスに日本語が含まれる場合は問題が発生することがあります（英数字のパスを推奨）

### アプリが起動しない

```bash
# node_modulesを削除して再インストール
rm -rf node_modules
npm install
npm start
```

### メタデータが正しく表示されない

- ファイルのID3タグが正しく設定されているか確認（Mp3tagなどのツールで編集可能）
- タグがない場合はファイル名がタイトルとして使用されます
