## 🎵 楽曲の追加

### ドラッグ＆ドロップ（推奨）

1. Finder でオーディオファイルまたはフォルダを選択
2. 左パネルの **ドロップゾーン**（「MP3 / M4A ファイルをここにドロップ」）へドラッグ
3. メタデータ（タイトル・アーティスト・カバーアート）が自動読み込みされプレイリストに追加

> 💡 **フォルダごとドロップ**すると、フォルダ内の全オーディオファイルを一括追加できます

### 対応フォーマット

`.mp3` `.m4a` `.aac` `.flac` `.wav` `.ogg`

## ▶ 再生操作

| ボタン | 動作 |
|--------|------|
| ▶ / ❚❚ | 再生 / 一時停止 |
| ⏮ | 前の曲（3秒以上再生中は先頭へ戻る） |
| ⏭ | 次の曲 |
| ↺ `10` | 10秒巻き戻し |
| ↻ `10` | 10秒早送り |
| 🔁 | リピートモード切り替え（後述） |
| 🔀 | シャッフルON/OFF |

**シークバー:** クリックまたはドラッグで再生位置を変更できます。

## 📂 プレイリストの作成・管理

### プレイリストを作成・保存する手順

```
① 音楽ファイルをドロップゾーンにドラッグ（複数可）
      ↓
② 左パネルに曲一覧が表示される
      ↓
③ 左パネル下部の 💾 保存 ボタンをクリック
      ↓
④ ダイアログにプレイリスト名を入力（例: My Rock Playlist）
      ↓
⑤ 「保存」ボタン または Enter キーで確定
      ↓
⑥ 「📂 SAVED PLAYLISTS」に保存される（アプリを閉じても保持）
```

### 保存済みプレイリストをロードする

1. 左パネルの **「📂 SAVED PLAYLISTS」** セクションを確認
2. ロードしたいプレイリストの **▶ ボタン** をクリック
3. そのプレイリストがキューに読み込まれ、自動的に再生開始

### プレイリストから曲を削除する

1. 削除したい曲をクリックして選択（ハイライト表示される）
2. 左パネル下部の **🗑 削除** ボタンをクリック
3. 選択中の1曲のみが削除され、次の曲が自動的に選択・再生される

### 保存済みプレイリストを削除する

「📂 SAVED PLAYLISTS」内の各プレイリスト右側の **✕ ボタン** をクリック

> ⚠️ 保存済みプレイリストのデータは `localStorage` に保存されているため、アプリを再起動しても保持されます。

## 🔁 ループ・リピート再生

中央パネルの **🔁 リピートボタン** をクリックするたびに切り替わります：

| バッジ表示 | モード | 動作 |
|-----------|--------|------|
| なし（暗い） | OFF | ループなし（最後の曲で停止） |
| `1` | 1曲リピート | 現在の曲を繰り返し再生 |
| `∞` | 全曲ループ | プレイリスト全曲を繰り返し再生 |

## 🔊 音量・速度

### 音量

- **起動時:** macOS のシステムボリュームが自動的に反映されます
- 中央パネル下部の **VOLUMEスライダー** で手動調整（0〜100%）

### 再生速度

**プリセット:** `0.5x` `0.75x` `1.0x` `1.25x` `1.5x` `2.0x`

**カスタム速度:**
1. 数値入力欄に `0.1` 〜 `4.0` を入力
2. `SET` ボタンまたは `Enter` で適用

## 📊 レベルメーター

再生中の音量レベルをL（左）・R（右）チャンネル別にリアルタイム表示します。

| 色 | 範囲 | 意味 |
|----|------|------|
| 🔵 青 | 0〜70% | 適正レベル |
| 🟡 黄 | 70〜90% | やや高いレベル |
| 🔴 赤 | 90〜100% | クリッピング危険域 |

> ⚠️ 赤ゾーンが頻繁に点灯する場合は音量を下げてください。

## 🎚 イコライザー（EQ）

右パネルの **「EQ」タブ** で設定します。

**起動時: Rock プリセットが ON 状態で適用されます。**

### プリセット一覧

| プリセット | 特徴 |
|-----------|------|
| **Flat** | フラット（補正なし） |
| **Rock** ← デフォルト | 低域と高域を強調 |
| **Pop** | 中域を強調 |
| **Jazz** | 低域と高域をやや強調 |
| **Classical** | 低域と高域を強調（繊細） |
| **Voice** | 中高域（声）を強調 |
| **Bass↑** | 低域を大幅強調 |
| **Heavy Metal** | 低域・中高域を強烈に強調 |

### 5バンドスライダー

| Band | 周波数 | 音域 |
|------|--------|------|
| 1 | 60 Hz | 超低域（サブベース） |
| 2 | 250 Hz | 低域（ベース） |
| 3 | 1 kHz | 中域（ボーカル） |
| 4 | 4 kHz | 中高域（存在感） |
| 5 | 16 kHz | 高域（空気感） |

範囲: **−12dB 〜 +12dB**  
「リセット」ボタンで全バンドを 0dB に戻せます。

| **Studio** | スタジオ録音風 |

## ⌨️ キーボードショートカット

| キー | 動作 |
|------|------|
| `Space` | 再生 / 一時停止 |
| `←` | 10秒巻き戻し |
| `Shift + ←` | 30秒巻き戻し |
| `→` | 10秒早送り |
| `Shift + →` | 30秒早送り |
| `↑` | 音量 +5% |
| `↓` | 音量 -5% |
| `N` | 次の曲 |
| `P` | 前の曲 |

> テキスト入力欄にフォーカス中はショートカットが無効になります。

## 🔧 トラブルシューティング

### 音が出ない
1. macOS のシステム音量を確認
2. アプリ内の VOLUME スライダーを確認
3. 一度停止 → 再生を試す
4. アプリを再起動

### ファイルが追加されない
- 対応フォーマットか確認（MP3/M4A/AAC/FLAC/WAV/OGG）
- ファイルパスに特殊文字が含まれていないか確認

### 曲をクリックしても再生されない
- プレイリストが空でないか確認
- 一度別の曲をクリックしてから試す

### アプリが起動しない
```bash
rm -rf node_modules
npm install
npm start
```

### 保存済みプレイリストが消えた
- `localStorage` はアプリのデータディレクトリに保存されています
- Electron のキャッシュをクリアすると消えることがあります
